VHDL Compliance Tests
#####################

Compliance-Tests is a collection of test suites to evaluate the support of `VHDL 2008 <https://standards.ieee.org/standard/1076-2008.html>`__
and `VHDL 2019 <https://standards.ieee.org/standard/1076-2019.html>`__ features.
